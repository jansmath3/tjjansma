Summary:

This was an HTML5 build for a client who wanted horizontal scrolling and lightbox representations of work.

Requirements:
  1. Lightweight (quick loading)
  2. Horizontal Scroll
  3. Logical/easy access to portfolio pieces
Problem:

How do I easily organize a portfolio so the intended audience can easily view a wide breadth of audio, video and photo work?

Solution:

Design was an import part of this project and psds were provided for me to code and improve upon. Early on in the process I decided HTML/CSS with some jquery would work great for the layout and lightboxes. Overall, this was a straightforward build that did not cause many problems. The only difficulty arose in encoding the videos in a manner that was light weight enough to run smoothly on a browser and could be used with a lightbox. Ultimately, I used lightweight fps .mov files in the lightboxes for the videos.

Outcome:

This website turned out really well and was used for several years. The client was happy with the outcome, but has since rebuilt the site himself for practice and to reflect his newest work.
